Millefeuille

Projet Web App en cours

(HTML / CSS / JS)
Bibliothèque Materialize et JQUERY

Intégration de :
- Plusieurs écrans,
- Champs de saisie (obligatoires), 
- Boutons pour afficher un autre écran,
- Boutons à choix,
- Boutons pour lancer une ou plusieurs actions,
- Sons et visuels
- Etc...

Millefeuille est une web app qui permet à son utilisateur de s'identifier et de choisir ses recettes. Les recettes sont détaillé étapes par étapes.
Les taux d'ingrédiants dans les recettes changent en fonction du nombre de personne choisi (2 ou 4).
