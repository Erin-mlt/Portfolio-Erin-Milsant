Interface web de voyage personnalisable avec une intégration d'IA

(HTML / CSS / JS)


Voici le projet « myslider » sur le thème d’une expérience personnalisée de voyages.
Le site « Voyage Vibe’s » permet de choisir son expérience personnalisée en découvrant des paysages de différents pays du monde (Chine, France, Espagne, Grèce, etc..).

Interface web comprenant :
- La présentation du site est accessible depuis la page « Accueil » qui intègre en plus, un bouton pour se rendre dans la partie personnalisable du site.
- La personnalisation est disponible depuis la page « personnalisation » et permet de sélectionner une ou plusieurs images pour animer un slider dans l’en-tête.
- La dernière page créée est celle du « Chat-IA » créé pour discuter avec un chatbot. Il permet au visiteur de poser ses potentielles questions.

Intégration d'éléments :
- En-tête personnalisable (slider si plusieurs images),
- Un grid contenant toutes les images de pays
- Chat IA (modèle d’IA : « gpt-5-nano »),

Préproduction :
- Wareframe du site,
- Planche tendance,
- Logos.
