Le fichier .jdf est un fichier standard XML utilisé pour décrire un travail d’impression.
Il contient toutes les infos nécessaires pour qu’une machine ou un logiciel sache quoi faire et comment le faire.
Exemple d’infos dans un JDF : format du papier, nombre d’exemplaires, couleurs utilisées, type d’imposition, finitions (pliage, reliure…), etc.
